# wangyi-project

## Project setup
```
npm install
```
## Project start serve
```
cd NeteaseCloudMusicApi

node app.js
```
## Run Project
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
